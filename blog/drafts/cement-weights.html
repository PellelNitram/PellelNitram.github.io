Cement weights

  • Sa 16 Dezember 2017
  • misc
  • #3D Printing, #Fluid Dynamics, #Edinburgh

Amongst others, I developed a code for simulating Newtonian fluids such as water for my Master studies. Choosing proper settings, one can easily generate large amounts of 3D data that resembles a turbulent flow in a box.

Since I am experienced in 3D printing and modeling, I took a snapshot of this data and 3D printed it after post-processing the data in Blender. The 3D print is post-processed with sandpaper and spray primer to smooth out the print layer structure. Finally, a silicone mold is created from the 3D print and a resin cast is produced from the silicone mold.

The steps that I performed are the following ones:

  1. Generate the turbulent data in a channel flow geometry. The boundary conditions are periodic in spanwise and streamwise directions. The wall-normal direction uses free-slip boundary conditions.
  2. The data is exported to the VTR format which is read by ParaView. From there, it is exported as STL. ParaView is used to find a nice iso surface in the time series.
  3. The raw simulation data STL is post-processed to generate a 3D print from it. I use Blender for that purpose.
  4. The post-processed STL is 3D printed.
  5. The 3D print is coated with a fill primer to hide the 3D print specific layer structure.
  6. A silicone mold is made from the 3D print which is used for a resin cast.

TD;DR: Please refer to here to see the final result.

The data is acquired by running the simulation that I wrote for a sufficiently long time in the incompressible Navier-Stokes equations’ turbulent regime at Re=400. My simulation is based on a pseudo-spectral algorithm so that one can reconstruct the full velocity fields from the spectral expansion coefficients that are saved during the simulation run. The fields are reconstructed and saved as VTR (using PyEVTK [TODO: link]) files to be able to inspect them against simulation time in ParaView. After deciding upon one flow field and one iso surface, the spectral coefficients are used to generate a high-resolution spatial field to get a nicer 3D print out of it. This data is finally saved as STL.

The STL file is post-processed in Blender: The 2D iso surface is extruded downwards to obtain a true 3D structure that can be 3D printed.

This model is 3D printed in PLA. I tested multiple sizes and multiple vertical resolutions on a UltiMaker 3. After holding all the 3D prints in my hand I decided to pick the highest vertical resolution of 0.6mm and a size of 9mm x 6mm x 3mm. This model is sanded down and white fill primer is sprayed onto it. It turns out that this laster step reduces the visible vertical layers quite sufficiently. That procedure might be an alternative to using ABS with an acetone bath in case ABS cannot be used.

A silicone mold is made in a small paper box. Using release spray is recommended to avoid destroying the silicone mold after removing the 3D print.

This silicone mold is finally used for a resin cast. One can use the silione mold as many time as it is still intact.
