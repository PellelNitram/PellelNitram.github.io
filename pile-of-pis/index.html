Pile of Pis

A stackable simple-to-build single board computer storage that is open-source and modular which is made from sustainable materials.

Perfectly suitable for Raspberry Pis, hard drives, and others.

Show features Get files

Stackable

Set up a pile of, say, Raspberry Pis by stacking Pile of Pi modules.

Modular

Combine single board computers with other equipment using adapters.

Simple

Fully customise your Pile of Pis setup with only a few parts.

Use cases

HDDs for NAS, multiple Raspberry Pis or additional electronical components? No problem!
